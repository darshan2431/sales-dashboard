# Sales Business Dashboard

An interactive sales analytics dashboard built with vanilla JavaScript and Chart.js. Visualizes revenue trends, regional performance, top products, and recent orders — with real-time region filtering.

## Live Demo

> Open `index.html` directly in any browser — no server needed.

## Features

- **Revenue trend chart** — monthly comparison of 2025 vs 2024 sales
- **Regional breakdown** — doughnut chart showing revenue split across South, North, West, and East
- **Top products** — ranked progress bar visualization by revenue contribution
- **Recent orders table** — order status with color-coded badges (Delivered / In transit / Delayed)
- **Region filter** — click any region to update all metrics and charts dynamically
- **Dark mode support** — automatically adapts to system preference

## Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Structure and semantic markup |
| CSS3 | Responsive layout, dark mode via `prefers-color-scheme` |
| JavaScript (ES6) | DOM manipulation, filter logic, dynamic chart updates |
| [Chart.js 4.4](https://www.chartjs.org/) | Bar chart and doughnut chart rendering |
